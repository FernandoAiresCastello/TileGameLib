GETTING STARTED

This page will guide you through the steps necessary to install TileGameLib in your local environment and show you how to start making games with it.

Note: This guide is for using the pre-built TGL library in your own projects. To build TGL in your own development environment you'll need to follow a different setup process which is not described in this page.

PREREQUISITES

  1. Visual Studio Community: If you don't have Visual Studio installed, you can click here to download and install it for free. TGL has been tested on the Community Edition of VS 2019 and 2022, so it should at least work in either of those.
  2. SDL 2: If you don't have the SDL 2 library installed, first you'll need to download and install it. TGL has been developed and tested only with SDL version 2.0.8. You can click here to download it from the official SDL repository on GitHub. If this link is dead for some reason just remember you need to download specifically the "SDL2-devel-2.0.8-VC.zip" file, so you can search for this on Google and you should find it somewhere else. Extract the downloaded ZIP file into your preferred location (recommended in the root C:/). Upon extracting it you should see a folder named "SDL2-2.0.8". SDL is now installed on your system.
  3. TileGameLib: Download the latest build of TGL.lib from the GitHub repository here.

SETUP

  1. Create a new empty C++ project in Visual Studio.

  2. Enter a name for your project, for example "HelloWorld", and the path where it will be created, then click on the "Create" button.

HELLO WORLD!

But first of all, let me show you a simple program that prints "Hello World!" on a window using TGL, to demonstrate how easy it is:

#include <TGL.h> int main(int argc, char* argv[]) { TGL tgl; tgl.window(0x0000ff); tgl.color_single(0xffff00); tgl.print_free("Hello World!", 35, 65); return tgl.halt(); }

Now let's explain what happens in each line:

  1. The TGL.h header file is included at the top.
  2. Inside the main function, the first line creates the TGL singleton. This initializes the internal state of the library.
  3. Next, the window() function opens the application window and sets its background color to blue.
  4. Next, the color_single() function enables the "single" color mode and sets the color of the index 1 for all subsequent tiles to yellow.
  5. Next, the print_free() function draws the string "Hello World!" on the screen at the x and y coordinates of 35 and 65 respectively.
  6. Lastly, the halt() function stops execution of the program, but keeps the window open and refreshing automatically so that you can see the output. Closing the window will terminate the application.

TileGameLib (TGL) is a C/C++ static software library built on top of SDL 2. TGL provides a simplified and minimalistic API for creating & drawing tiles, windowing, text printing, keyboard & mouse input, sound, timers, simple collision detection, filesystem access and other conveniences. Graphically, it aims to somewhat resemble the look and feel of Nintendo GameBoy Color games and other 8-bit tile-based computer systems.
